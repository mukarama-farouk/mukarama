This Python program contains a simple program for converting currency in real-time using the Forex Python Library. The Forex-Python library provides the most direct way to get a currency conversion rate through API calls. It has different functions, which take the currency codes you want to convert from one currency to another and then return the current value. In this Python program, we are converting USD to INR. It will ask you to input the amount and the currency codes you want to convert. An example of currency conversion using forex_ python:
